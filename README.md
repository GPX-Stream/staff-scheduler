# Staff Scheduler

A React-based staff scheduling application for planning weekly staff coverage across multiple timezones.

## Features

- Visual weekly schedule grid (7 days x 24 hours)
- Multi-timezone support with automatic conversion
- Drag-and-drop shift scheduling
- Staff visibility toggles
- PDF export
- Edit/View mode toggle

## Getting Started

```bash
npm install
npm run dev
```

## Deployment

This project is configured for deployment on Vercel. Simply connect your GitHub repository to Vercel for automatic deployments.
